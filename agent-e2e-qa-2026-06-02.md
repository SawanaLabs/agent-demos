# Agent Demo E2E QA - 2026-06-02

Scope: end-to-end smoke QA for ready demo pages in `apps/web`, using the deployed production domain in the browser.

Rules:

- Read-only with respect to code/config. The browser may create visitor-scoped chats, cookies, localStorage, DB rows, and model usage while sending messages.
- Exercise core behavior by clicking sample suggestions first, then sending follow-up turns such as "continue" where the demo supports chat continuation.
- Record reproducible failures with exact URL, steps, expected behavior, actual behavior, visible error text, relevant network/console details, and severity.
- Avoid destructive UI actions such as deleting chats, clearing history, or uploading private/local files.
